JavaScript运算符优先级

Licc4/8/2021, 10:11:48 AMJavaScriptJavaScript

运算符的优先级决定了表达式中运算执行的先后顺序，优先级高的运算符最先被执行。

# 示例

A && B
// 若A为true，则return B。若A为false，return A

A || B
// 若A为true，return A。若A为false，则return B

false || 3 && 2
// ||的左侧为false，所以返回(3 && 2)的结果；
// 又，3为true，所以直接返回2
